# Términos de Uso — Syncora

> Lee con atención antes de utilizar Syncora. Al instalar o usar el software, aceptas estos términos.

## 1. Acerca de Syncora

Syncora es una aplicación **open source** distribuida bajo la licencia **MIT** (ver `LICENSE`). Permite buscar, descargar y sincronizar subtítulos de archivos de vídeo usando proveedores externos.

## 2. Uso del software

- Syncora se proporciona **"tal cual"**, sin garantías de ningún tipo, expresas o implícitas.
- Eres el único responsable del uso que haces del software y del contenido que procesas con él.
- El uso del software es **gratuito** y voluntario.

## 3. Proveedores de subtítulos

Syncora se integra con servicios de terceros para buscar y descargar subtítulos, incluyendo, entre otros:

- **SubDL**
- **OpenSubtitles**
- **SubSource**

Al usar estos proveedores, aceptas respetar los **términos de uso de cada proveedor**. Las API keys, cuentas y límites de uso son responsabilidad tuya.

Los desarrolladores de Syncora **no controlan**:

- La disponibilidad de los proveedores
- El contenido de los subtítulos
- Cambios de precio, límites o políticas de los proveedores

## 4. Contenido y derechos de autor

- Eres responsable de asegurarte de que el uso de los subtítulos descargados sea **legal** en tu jurisdicción.
- Syncora no aloja, distribuye ni selecciona el contenido de los subtítulos — solo facilita la búsqueda y descarga desde fuentes públicas de terceros.
- Respeta los derechos de autor de obras audiovisuales, subtítulos y traducciones.

## 5. Dependencias externas

Syncora descarga y gestiona localmente las siguientes herramientas, obtenidas de sus fuentes oficiales:

- **ALASS** — para sincronización de subtítulos
- **FFmpeg / FFprobe** — para embeber softsubs (opcional)

Estos programas tienen sus propias licencias. Consulta sus repositorios para más detalles.
